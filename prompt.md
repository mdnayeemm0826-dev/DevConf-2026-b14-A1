# Prompt: DevConf 2026 — FAQ / Logistics Section

## Context
I'm building a landing page for **DevConf 2026**, a developer conference. The page already has a dark navy hero, a light "Meet the Speakers" card grid, and a pricing section. I need a simple **FAQ / Logistics** section, written as an absolute beginner would — no advanced JavaScript, no frameworks, no complex animations.

## Task
Design and build a basic **FAQ accordion** section covering: parking, Wi-Fi, code of conduct, dietary options, and refund policy.

Requirements:
1. The accordion open/close behavior should work with **zero JavaScript**
2. A simple `+` / `−` icon indicator that flips when a question is opened (pure CSS, no icon library required)
3. Light background section (not dark), keeping it visually simple and easy to understand
4. Basic accessible focus state on the question when tabbed to with keyboard
5. Clean, beginner-readable HTML and CSS — no deep nesting, no advanced selectors, no build tools

## Style requirements
- Keep the structure minimal: a heading, a short subtitle, then a list of question/answer items
- Use plain class names, external CSS only (no inline styles)
- Sizing and spacing should be simple, sensible defaults — nothing overly precise or complex

## Deliverable
One HTML block for the FAQ section, and a separate external CSS file, simple enough for a beginner to read and modify.
